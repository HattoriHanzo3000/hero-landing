# B2 Beruf screenshots (by site language)

Place **6** WebP files per locale (~500×1125 px). The gallery uses the same language as the page (EN / DE switcher).

## English

```
public/apps/b2-beruf/screenshots/en/01.webp
public/apps/b2-beruf/screenshots/en/02.webp
… through 06.webp
```

## German

```
public/apps/b2-beruf/screenshots/de/01.webp
public/apps/b2-beruf/screenshots/de/02.webp
… through 06.webp
```

Use the **same screen order** in both folders (e.g. 01 = chapter overview, 02 = word list, …).

Alt text: `lib/i18n/dictionaries/en.ts` and `de.ts` → `b2Beruf.landing.screenshotAlts`.
