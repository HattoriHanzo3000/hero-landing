# Einbürgerungstest screenshots (by site language)

Place **6** WebP files per locale (~500×1125 px). The gallery uses the same language as the page (EN / DE switcher).

## English

```
public/apps/einbuergerungstest/screenshots/en/01.webp
public/apps/einbuergerungstest/screenshots/en/02.webp
… through 06.webp
```

## German

```
public/apps/einbuergerungstest/screenshots/de/01.webp
public/apps/einbuergerungstest/screenshots/de/02.webp
… through 06.webp
```

Use the **same screen order** in both folders (e.g. 01 = topic overview, 02 = question list, …).

Alt text: `lib/i18n/dictionaries/en.ts` and `de.ts` → `citizenship.landing.screenshotAlts`.
